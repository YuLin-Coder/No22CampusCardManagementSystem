> 注意： 界面出现乱码的话，环境变量中添加-Dfile.encoding=GB18030
> 
>------------------------------------------------------------------------------------------------------------------------
>一. schoolcard.txt文档相等于校园卡的数据表.

    第一列: 校园卡主键
    第二列: 校园卡密码
    第三列: 校园卡余额
    第四列: 校园卡挂失(1:挂失,0:未挂失)
    第五列: 校园卡用户名
    第六列: 校原卡是否已删除(1:删除,0:未删除)

>------------------------------------------------------------------------------------------------------------------------
>二. credit.txt文档相等于充值的数据表.

    第一列: 充值记录主键
    第二列: 校园卡主键
    第三列: 校园卡充值金额
    第四列: 校园卡充值时间

------------------------------------------------------------------------------------------------------------------------
>三. consume.txt文档相等于消费的数据表.

    第一列: 消费记录主键
    第二列: 校园卡主键
    第三列: 校园卡消费金额
    第四列: 校园卡消费时间
>------------------------------------------------------------------------------------------------------------------------
